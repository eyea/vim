# vim
This is eyea's vim files.

### How to use vim
1. Copy `vimrc` (this repo includes) to your directory `~/`
2. [Setup Vundle](https://github.com/VundleVim/Vundle.vim#quick-start)
```js
git clone https://github.com/VundleVim/Vundle.vim.git ~/.vim/bundle/Vundle.vim
```
3. cmd
```js
$ vim
// Install Vundleï¼šgit clone http://github.com/gmarik/vundle.git ~/.vim/bundle/vundle
// Install Pluginsï¼š :BundleInstall
// Update Pluginsï¼š :BundleInstall!
// Unreload the Plugins not in the lists(.vimrc)ï¼š :BundleClean"
```
